html-верстальщик

что такое html-верстка?

HTML-верстка — это расположение текста, заголовков и изображений на странице. Верстальщик создаёт основу и элементы веб-страницы, указывает их расположение, затем оформляет её в соответствии с дизайном. От качества верстки страницы зависит её загрузка, скорость открытия и корректность отображения в браузере.

Для создания используются специальные языки — HTML, CSS, JavaScript. Структура страницы собирается с помощью языка HTML (HyperText Markup Language — язык гипертекстовой разметки). CSS (Cascading Style — каскадные таблицы стилей) придаёт стиль будущей странице. А JavaScript (динамический язык программирования) придает интерактивности веб-страницам.
